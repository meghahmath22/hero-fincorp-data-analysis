# Hero FinCorp Data-Driven Analysis

## 📌 Project Overview

This project analyzes Hero FinCorp’s loan portfolio, customer behavior, and branch performance to identify default risks, improve efficiency, and enhance profitability.

---

## 📊 Key Analysis Performed

* Data Cleaning & Preparation
* Exploratory Data Analysis (EDA)
* Default Risk Analysis
* Branch Performance Evaluation
* Customer Segmentation
* Profitability Analysis

---

## 🔍 Key Insights

* Low credit score customers have higher default probability
* High EMI burden increases risk of default
* Certain branches show high disbursement but poor recovery
* Customer segmentation helps identify high-value vs high-risk groups

---

## 💡 Recommendations

* Implement stricter credit score thresholds
* Optimize EMI structure for risky customers
* Improve recovery strategies in low-performing branches
* Use segmentation for targeted loan offerings

---

## 🛠️ Tools Used

* Python (Pandas, NumPy)
* Matplotlib, Seaborn
* Jupyter Notebook

This project demonstrates how data-driven decision-making can reduce loan defaults, improve operational efficiency, and increase profitability.
